# Car-sales-analysis-Dashboard
Car sales analysis Dashboard
# Overview

The purpose of this project is to create a comprehensive dashboard for analyzing car sales performance. This dashboard helps users track key indicators such as total sales, average price, cars sold, sales breakdown by body style, color, company, and dealer regions. Interactive features like filters and slicers allow in-depth drill-down and exploration of the data.

# Project overview

* Developed an interactive Power BI dashboard to visualize and analyze car sales data from multiple dimensions.

* Used advanced filters, user-driven parameters, and DAX calculations to allow segmented analysis on factors like body style, engine type, transmission, dealer region, and more.

* Implemented various visualizations including time trends, donut charts, bar graphs, and detailed tables for deeper insights.

* Enables stakeholders to identify sales patterns, top-performing regions, and compare company-wise trade performance.

* Supports decision making by providing actionable sales and inventory insights for automotive dealers and managers.

* Integrates multiple data tables for comprehensive sales tracking.

* Lets users filter data by body style, transmission, engine type, dealer region, and more for precise analysis.

* Visualizes sales performance with interactive line charts, donut charts, bar charts, and tables.

* Displays summary KPIs including year-to-date sales, average price, and cars sold to track business health.

* Offers insights into sales trends weekly and by product details to spot growth or decline patterns.

* Breaks down sales by color, body style, and dealership, helping identify top-selling variants and locations.

* Presents company-wise sales comparison by both value and volume for competitive benchmarking.

* Provides gender-based analysis of buyers per dealer to understand customer demographics.

* Enables stakeholders to make data-driven decisions in pricing, inventory, and marketing.

* Dashboard design supports real-time updates and easy customization for different data sources.
